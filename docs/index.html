What's this

This is a collection of column-staggered and ortholinear ergonomic keyboards.

It is not as large as some other collections, but you can sort and filter the keyboards.

"The rules"
  • Column-staggered and ortholinear keyboards only.
    This avoids some ambiguity with counting columns and similar.
  • Keyboards have to be buildable or obtainable in some way - no teasers.
Where else to look for keyboards
